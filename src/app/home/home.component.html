@ngx-tour

Product tour using Angular (2+)

About

This is a product tour library built with Angular (2+). It's inspired by angular-ui-tour.

Implementations

The core functionality of this tour package is implemented in @ngx-tour/core ().

There are multiple implementations of the @ngx-tour:

  • @ngx-tour/md-menu
  • @ngx-tour/ng-bootstrap
  • @ngx-tour/ngx-bootstrap
  • @ngx-tour/ngx-popper
  • @ngx-tour/console
NOTE: TourService is exported by @ngx-tour/core and each of the implementations export a more specialized version. When using @ngx-tour across multiple modules, ensure that your imports all reference the implementation-specific TourService or you will get two different instances of TourService injected and each will have a different configuration/steps/etc...
